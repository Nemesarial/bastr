# Usage
<!-- usage -->
```sh-session
$ npm install -g bastr
$ bs COMMAND
running command...
$ bs (-v|--version|version)
bastr/0.1.0 darwin-x64 node-v10.14.2
$ bs --help [COMMAND]
USAGE
  $ bs COMMAND
...
```
